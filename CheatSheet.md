Commands
When prompted with letter or number choices:
- use All lowercase letters
    - i.e: Which option? (a/b/c) ==> enter a, b, or c
- don't use any other characters besides the options presented
    - i.e: Which option? (1/2/3) ==> enter 1, 2, or 3

When prompted with destination choices:
- use EXACT spelling
    - i.e: Where to go? (Art Museusm or Tyler) ==> enter "Art Museum" or "Tyler" (without quotation marks)
